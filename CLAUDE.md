## Project Overview

PyTo Code is a Python-first Code Agent framework - a lightweight, extensible vertical agent framework for code generation, execution, and tooling with LLMs. Currently in early development (v0.1.0).

## Status

This is a new project with minimal initial structure. The framework architecture, build system, and test infrastructure have not yet been established. When implementing new features:

- Follow Python best practices for agent/LLM frameworks
- Include proper type hints and docstrings
- Add tests using pytest
- Use a modern Python project structure (pyproject.toml recommended)
